# Excel-Data-Cleaning-And-Transformation-Project
Data cleaning and transformation using Excel (handling missing values, formatting, conditional formatting)

Overview

This project focuses on cleaning and transforming raw dataset using Excel.
1. Handled missing values using AVERAGEIFS and logical imputation
2. Standardized inconsistent text formats
3. Removed duplicate records
4. Split and merged columns
5. Applied currency and date formatting
6. Used conditional formatting (data bars & category highlighting)

 Tools Used
Microsoft Excel

 Outcome

Cleaned and structured dataset ready for analysis.
